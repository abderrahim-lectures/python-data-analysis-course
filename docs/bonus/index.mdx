# 🎁 Choose your Capstone

**Congratulations on finishing the course.** Everything so far ran in a sandboxed, in-browser playground — so you could start writing Python on day one with zero setup. A Capstone project is the graduation step: install Python for real on your own machine, then build something the playground could never run.

This is optional and ungraded. A new Capstone project is added every year — you're free to pick this year's, or any past year's, whichever sounds more interesting to you. Finishing one is enough; finishing more than one is just for fun.
