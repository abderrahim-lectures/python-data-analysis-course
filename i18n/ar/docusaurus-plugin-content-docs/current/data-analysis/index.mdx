# 📊 Pandas وتحليل البيانات

## لماذا Pandas؟ مشكلة السرعة

إذا خضت المسار الصعب في Python 101، فأنت بالفعل قِست زمن كودك الخاص بلغة Python الخالصة لعدّ الكلمات
في مجموعة نصوص — وشعرت ببطئه كلما كبرت المجموعة. هذا ليس مصادفة: حلقات Python العادية بطيئة لهذا النوع
من العمل. توجد Pandas (المبنية على numpy) بالتحديد لحل هذه المشكلة، عبر تنفيذ الحلقات بكود C سريع
ومُتَّجَه (vectorized) تحت الغطاء بدلاً من Python نفسها. هذا القسم يتعلق بتعلّم استخدام تلك السرعة.

**🎯 أهداف التعلم** — بنهاية هذه الأسابيع الخمسة ستكون قادرًا على:
- تحميل واستكشاف وتنظيف بيانات جدولية باستخدام pandas
- تصفية وتجميع وتلخيص مجموعات بيانات حقيقية
- إعادة إنتاج تحليل بأسلوب Kaggle من البداية إلى النهاية
