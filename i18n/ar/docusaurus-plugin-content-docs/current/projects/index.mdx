# 🌍 مشاريع من العالم الحقيقي

كل شيء حتى الآن عمل في بيئة تجريبية معزولة داخل المتصفح — حتى تتمكن من البدء بكتابة Python من اليوم الأول بلا أي إعداد. هذه المشاريع هي خطوة التخرّج: ثبّت Python فعليًا على جهازك الخاص، ثم ابنِ شيئًا لم تستطع بيئة البرمجة تشغيله أبدًا.

هذه المشاريع اختيارية وغير مُقيَّمة. تصفّحها في أي وقت — تذكر مقدمة كل مشروع الخلفية المطلوبة (عادةً Python 101، وأحيانًا تحليل البيانات أيضًا)، لذا لست بحاجة لإنهاء الدورة بأكملها أولًا. تُضاف مشاريع جديدة كلما استحقت أداة أو إطار عمل رائج دليلًا عمليًا حقيقيًا — وليس وفق جدول زمني ثابت.
