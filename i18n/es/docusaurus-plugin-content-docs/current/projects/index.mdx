# 🌍 Proyectos del mundo real

Todo hasta ahora se ejecutó en un playground aislado dentro del navegador — para que pudieras empezar a escribir Python desde el primer día sin ninguna configuración. Estos proyectos son el paso de graduación: instala Python de verdad en tu propia máquina, y luego construye algo que el playground nunca pudo ejecutar.

Son opcionales y no calificados. Explóralos en cualquier momento — la introducción de cada proyecto indica qué conocimientos previos espera (normalmente Python 101, a veces también Data Analysis), así que no necesitas haber terminado todo el curso primero. Se añaden proyectos nuevos cada vez que una herramienta o framework de tendencia merece una guía práctica real — no según un calendario fijo.
