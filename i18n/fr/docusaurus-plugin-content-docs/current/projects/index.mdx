# 🌍 Projets concrets

Tout jusqu'ici tournait dans un bac à sable isolé, dans le navigateur — pour que vous puissiez commencer à écrire du Python dès le premier jour sans aucune configuration. Ces projets sont l'étape de remise de diplôme : installez Python pour de vrai sur votre propre machine, puis construisez quelque chose que le bac à sable n'a jamais pu exécuter.

Ils sont optionnels et non notés. Parcourez-les à tout moment — l'introduction de chaque projet précise les prérequis attendus (généralement Python 101, parfois aussi Data Analysis), donc vous n'avez pas besoin d'avoir terminé tout le cours d'abord. De nouveaux projets sont ajoutés dès qu'un outil ou framework tendance mérite un vrai guide pratique — pas selon un calendrier fixe.
