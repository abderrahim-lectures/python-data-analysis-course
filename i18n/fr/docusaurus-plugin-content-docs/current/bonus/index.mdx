# 🎁 Choisissez votre Capstone

**Félicitations d'avoir terminé le cours.** Tout jusqu'ici tournait dans un bac à sable isolé, dans le navigateur — pour que vous puissiez commencer à écrire du Python dès le premier jour sans aucune configuration. Un projet Capstone est l'étape de remise de diplôme : installez Python pour de vrai sur votre propre machine, puis construisez quelque chose que le bac à sable n'a jamais pu exécuter.

Ceci est optionnel et non noté. Un nouveau projet Capstone est ajouté chaque année — vous êtes libre de choisir celui de cette année, ou celui de n'importe quelle année précédente, celui qui vous semble le plus intéressant. Terminer un seul suffit ; en terminer plusieurs, c'est juste pour le plaisir.
