# 🐍 Python 101

**🎯 Objectifs d'apprentissage** — à la fin de ces 5 semaines vous serez capable de :
- Écrire des programmes Python et raisonner sur eux en utilisant les variables, le flux de contrôle, et les fonctions
- Travailler avec les structures de données de base de Python : listes, dicts, tuples, ensembles
- Lire et écrire des fichiers CSV — votre premier aperçu du travail avec de vraies données

Les deux parcours couvrent les 5 mêmes semaines. Choisissez celui qui correspond à votre façon d'apprendre :
