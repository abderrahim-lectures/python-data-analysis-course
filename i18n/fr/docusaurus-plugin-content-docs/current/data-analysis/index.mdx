# 📊 Pandas et analyse de données

## Pourquoi Pandas ? Le problème de la vitesse

Si vous avez suivi le parcours difficile de Python 101, vous avez déjà chronométré votre propre code de comptage
de mots en Python pur sur un corpus de texte — et l'avez senti ralentir à mesure que le corpus grandissait.
Ce n'est pas une coïncidence : les boucles Python ordinaires sont lentes pour ce genre de travail. Pandas
(construit sur numpy) existe précisément pour résoudre cela, en effectuant les boucles en code C rapide et
vectorisé sous le capot plutôt qu'en Python lui-même. Cette section porte sur l'apprentissage de l'utilisation
de cette vitesse.

**🎯 Objectifs d'apprentissage** — à la fin de ces 5 semaines vous serez capable de :
- Charger, explorer, et nettoyer des données tabulaires avec pandas
- Filtrer, grouper, et agréger de vrais jeux de données
- Reproduire une analyse de style Kaggle de bout en bout
